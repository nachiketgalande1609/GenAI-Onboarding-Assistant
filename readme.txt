Steps to Run the App:

1. Create a directory for the project and paste the files into that directory.
2. Navigate to the created directory using the terminal.
3. Install virtualenv by running the command: 
    `pip install virtualenv`
4. Create a virtual environment named 'venv' by running: 
    `virtualenv venv`
5. Activate the virtual environment: 
    `venv\Scripts\activate`
6. Install the required packages listed in the requirements.txt file by running: 
    `pip install -r requirements.txt`
7. Run the Flask application by executing the command: 
    `python app.py`
8. Once the application is running, open a web browser and go to the address provided by Flask, typically `http://127.0.0.1:5000/`.
9. You can terminate the application by pressing `Ctrl + C` in the terminal.
